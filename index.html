Utility-First CSS Framework

Welcome to our comprehensive **style guide** and **cheat sheet**. This page demonstrates all design tokens (colors, spacing, fonts, shadows) and provides a reference table for every major utility class in the framework.

1. Colors

We provide 11 main colors, each with a light and dark variant, plus a grayscale range. These are defined as CSS variables (e.g., --color-1, --color-1-light, --gray-100, etc.) and used within utility classes (like .bg-color-1, .text-color-1-dark, .border-gray-300).

2. Spacing, Fonts, and Shadows

Spacing Scale

We define 10 spacing steps: --spacing-1 (0.25rem) up to --spacing-10 (6rem). These power margin/padding utilities, e.g. .m-4, .p-2, .mx-8.

  • --spacing-1: 0.25rem (4px)
  • --spacing-2: 0.5rem (8px)
  • --spacing-3: 0.75rem (12px)
  • --spacing-4: 1rem (16px)
  • --spacing-5: 1.5rem (24px)
  • --spacing-6: 2rem (32px)
  • --spacing-7: 2.5rem (40px)
  • --spacing-8: 3rem (48px)
  • --spacing-9: 4rem (64px)
  • --spacing-10: 6rem (96px)

Font Families

The variables --font-sans, --font-serif, and --font-mono define our default typefaces. Utility classes (.font-sans, .font-serif, .font-mono) let you switch easily.

Radius & Shadows

Border radius tokens (e.g. --radius-sm, --radius-md, etc.) can be used in classes like .rounded-sm, .rounded-lg. Similarly, shadow tokens (--shadow-sm, --shadow-md, --shadow-lg) power shadow utilities like .shadow-sm, .shadow-lg.

3. Typography Demo

Our framework includes font-size utilities (.text-xs to .text-9xl), line-height utilities (.leading-tight to .leading-loose), letter-spacing (.tracking-tight, .tracking-wide), transforms (.uppercase, .capitalize), and font weights (.font-normal, .font-bold, etc.).

4. Cheat Sheet – Major Utility Classes

Below is a broad listing of our major utility classes. They are grouped by category for convenience. For an even more exhaustive reference, please see compiled.css.

Class Description CSS Declaration
Spacing (Margin & Padding)
.m-1 to .m-10 Margin (all sides) steps 1–10 margin: var(--spacing-N);
.p-1 to .p-10 Padding (all sides) steps 1–10 padding: var(--spacing-N);
.mx-1 / .mx-2 / etc. Horizontal margin (left + right) margin-left: var(--spacing-N); margin-right: var(--spacing-N);
.my-1 / .my-2 / etc. Vertical margin (top + bottom) margin-top: var(--spacing-N); margin-bottom: var(--spacing-N);
.-m-1 to .-m-10 Negative margins margin: calc(var(--spacing-N) * -1);
.ml-auto, .mr-auto Auto margin for left or right margin-left: auto; / margin-right: auto;
Sizing (Width & Height)
.w-1/2, .w-1/3, etc. Fractional widths width: 50%; or width: 33.3333%;
.w-4, .w-8, etc. Fixed width in rem steps (example scale) width: 1rem; / width: 2rem;
.h-screen Full viewport height height: 100vh;
.min-w-0, .max-w-full Min/Max width constraints min-width: 0; / max-width: 100%;
.w-fit, .h-fit Fit-content sizing width: fit-content;, height: fit-content;
Typography
.text-xs to .text-9xl Font size from ~12px up to ~128px font-size: 0.75rem; ... font-size: 8rem;
.leading-tight ... .leading-loose Line heights from 1.25 up to 2 line-height: 1.25; ... line-height: 2;
.uppercase, .lowercase, .capitalize Text transform utilities text-transform: uppercase; etc.
.font-sans, .font-serif, .font-mono Switch between the 3 main font families font-family: var(--font-sans); etc.
.font-thin ... .font-black Font weight from 100 up to 900 font-weight: 100; ... font-weight: 900;
.tracking-tight, .tracking-wide Letter spacing utilities letter-spacing: -0.02em; / 0.05em;
.text-left, .text-right, etc. Text alignment utilities text-align: left; / right; / center;
Colors (Background, Text, Border)
.bg-color-1 ... .bg-color-1-dark Background color from our 11 main color tokens background-color: var(--color-1);, etc.
.text-color-2, .text-color-2-light, etc. Text color from main color set color: var(--color-2-light); etc.
.bg-gray-100 ... .text-gray-900 Background or text color from the grayscale set background-color: var(--gray-100); / color: var(--gray-900);
.border-color-3-dark Border color from color-3 dark variant border-color: var(--color-3-dark);
Layout & Display
.block, .inline, .inline-block Display utilities display: block;, etc.
.hidden Hide element (display: none) display: none;
.overflow-hidden, .overflow-auto Overflow control overflow: hidden; or auto;
.container Centered container with max-width (1200px by default) margin: 0 auto; max-width: 1200px;
Flex
.flex Sets display: flex display: flex;
.flex-row, .flex-col Flex direction flex-direction: row; or column;
.items-center, .justify-between Flex item alignment & justification align-items: center;, justify-content: space-between;
.flex-wrap, .flex-nowrap Flex wrapping behavior flex-wrap: wrap;, etc.
.order-1, .order-last Flex item ordering order: 1;, order: 9999;
Grid
.grid Display: grid (defaults to 12 columns if set in the class) display: grid;
.col-6, .sm:col-4 Grid columns spanning 6 or 4 (responsive variants) grid-column: span 6;, etc.
.gap-4, .gap-2 Gap between grid items (pulls from spacing tokens) gap: var(--spacing-4);
Positioning & Z-Index
.relative, .absolute, .fixed Position utilities position: relative;, etc.
.top-0, .right-0 Offsets (top/right/bottom/left) set to 0 top: 0;, right: 0;
.z-50, .z-9999 Large jumps for z-index z-index: 50;, z-index: 9999;
Borders & Radius
.border, .border-2, .border-4 Border width border-width: 1px;, 2px;, 4px;
.border-solid, .border-dotted Border style border-style: solid; or dotted;
.rounded, .rounded-full Border radius (smaller or fully rounded) border-radius: 0.25rem;, 9999px;
.ring-2, .ring-primary Focus ring outlines box-shadow: 0 0 0 2px currentColor; or var(--color-1)
Backgrounds & Effects
.bg-cover, .bg-contain Background size background-size: cover; or contain;
.bg-no-repeat, .bg-repeat-x Background repeat settings background-repeat: no-repeat; or repeat-x;
.shadow-sm, .shadow-lg Box shadow utilities box-shadow: var(--shadow-sm); or var(--shadow-lg);
.opacity-50, .opacity-75 Opacity steps opacity: 0.5; or 0.75;
Transforms & Transitions
.translate-x-1/2 Translate element horizontally by 50% transform: translateX(50%);
.rotate-45 Rotate element by 45 degrees transform: rotate(45deg);
.scale-95 Scale element to 95% transform: scale(0.95);
.transition-all Apply a generic transition to all properties transition: all 0.2s ease-in-out;
.duration-300, .delay-150 Transition timing utilities transition-duration: 300ms;, transition-delay: 150ms;
Animations
.animate-spin Spin element indefinitely animation: spin 1s linear infinite;
.animate-pulse Pulse animation animation: pulse 2s ease-in-out infinite;
.animate-bounce Bounce animation animation: bounce 1s infinite;
Forms & Tables
.form-input, .form-select Basic styled input / select fields display: block; width: 100%; padding: var(--spacing-2); ...
.table-auto, .table-fixed Table layout utilities table-layout: auto; or fixed;
.table-bordered, .table-striped Border and striped row utilities border-collapse: collapse; / nth-child(odd) background
.table-hover Hover highlight on table rows tbody tr:hover { background-color: ... }
